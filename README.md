# Het Nagda — Portfolio

> Live at **[hetnagda20.github.io](https://hetnagda20.github.io)**

Personal portfolio website built as a single HTML file — no frameworks, no build step.

## Features

- **Multi-section navigation** — sidebar dot nav, scroll, swipe, and keyboard arrows
- **Terminal hero** — animated CLI intro that replays on every visit
- **D3.js radar chart** — interactive skill visualization
- **Animated skill bars** — triggered on section entry
- **Expandable projects** — click to reveal details + GitHub/demo links
- **Page view counter** — live hit counter via hits.sh
- **Mobile responsive** — touch swipe navigation

## Stack

Built with vanilla HTML, CSS, and JavaScript. No dependencies except:
- [D3.js](https://d3js.org/) — radar chart
- [Google Fonts](https://fonts.google.com/) — Syne, Lora, JetBrains Mono

## Local Development

No build step needed — just open the file:

```bash
git clone git@github.com:HetNagda20/HetNagda20.github.io.git
cd HetNagda20.github.io
open index.html
```
